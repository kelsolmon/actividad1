Investiga y desarrolla

1. Dato:

Es una representación simbólica de un atributo o característica de una entidad.

2. informacion:

Resultado del procesamiento de datos que proporciona significado y contexto.

3. Diferencia entre dato e información:

Los datos son crudos y sin procesar, mientras que la información es el resultado del procesamiento de esos datos.

4. Tipos de datos:

Incluyen numéricos, textuales, booleanos, fecha/hora, entre otros.

5. Modelado de Datos:

Proceso de crear un modelo de la estructura de una base de datos.

6. tipos de datos

Variabilidad: Estáticas (datos no cambian mucho) y dinámicas (datos cambian frecuentemente).

Contenido: Textuales, multimedia y mixtas.

Modelo de base de datos: Relacionales (tablas), NoSQL (flexibles).

7. Modelo Relacional:

Organiza datos en tablas con relaciones entre ellas.

8. Elementos del Diagrama Entidad-Relación:

Entidades (tablas), atributos (columnas), relaciones (conexiones) y cardinalidad (número de ocurrencias).

9. Entidad:

Objeto o concepto del mundo real representado como una tabla en la base de datos.

10. Atributo:

Característica de una entidad. Puede ser numérico, textual, etc.

11. Relación:

Asociación lógica entre entidades que representa cómo están conectadas.

12. Llaves primarias y foráneas:

Identifican de manera única registros en una tabla y mantienen integridad referencial entre tablas, respectivamente.

13. Reglas del Negocio:

Restricciones que definen aspectos del negocio, incluyendo restricciones en los datos almacenados.

14. Cardinalidad o mapeo:

Describe el número de ocurrencias de una entidad que pueden estar relacionadas con otra entidad (uno a uno, uno a muchos, muchos a muchos).
